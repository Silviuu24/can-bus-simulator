# CAN Bus Simulator

A virtual CAN Bus simulator implemented in Python.

## Overview
Simulates communication between multiple ECU nodes over a virtual CAN Bus.

## Nodes
- **ECU Engine** - sends engine temperature and RPM
- **ECU Dashboard** - receives and displays data

## Technologies
- Python 3.x
- python-can (virtual interface)
- threading, queue
